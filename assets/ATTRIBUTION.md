# Asset Attribution

This file tracks all external art/audio/font resources used by the project.

## Status

- Core visual placeholders have been replaced by imported CC0 art packs listed below.

## Required fields per imported pack

For every third-party asset pack, add:

1. Pack name
2. Author / creator
3. Source URL
4. License (exact)
5. Files imported into this repository
6. Any required attribution text

## Entry template

```md
### <Pack Name>
- Author: <Author Name>
- Source: <https://...>
- License: <CC0 / CC-BY-4.0 / OFL-1.1 / etc>
- Imported files:
  - `assets/...`
  - `assets/...`
- Attribution required: <yes/no>
- Attribution text:
  - "<exact required text>"
```
