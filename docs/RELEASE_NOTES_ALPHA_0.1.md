# Alpha 0.1

First public alpha for **Echoes of the Eternal** (Godot 4.2).

## What’s in this build

- Playable vertical slice: movement, HUD prompts, dialogue, journal, map transitions, save/load.
- Narrative source expansion (canon arc, characters, quests, dialogue hooks) and importer-generated runtime JSON.
- Lore discovery persisted in saves; journal lists discovered lore.
- Linux x86_64 export (binary + external `.pck`).

## How to run (Linux)

1. Download `echoes-of-the-eternal.x86_64` and `echoes-of-the-eternal.pck`.
2. Put both files in the **same folder**.
3. `chmod +x echoes-of-the-eternal.x86_64`
4. Run: `./echoes-of-the-eternal.x86_64`

## Controls

- Move: WASD / arrows / left stick  
- Interact: E / Space / gamepad south  
- Attack: J / Ctrl / gamepad west  
- Journal / menu: Esc / Start
